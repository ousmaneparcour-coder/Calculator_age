Age Calculator App

Application web qui calcule l'âge exact (années, mois et jours)
à partir d'une date de naissance. Réalisée dans le cadre du TP
de développement web.

 Aperçu

Challenge : [Frontend Mentor — Age Calculator App](https://www.frontendmentor.io/challenges/age-calculator-app-dF9DFFpj-Q)

 Fonctionnalités

- Calcul précis en années, mois et jours
- Validation des champs (date invalide, date future)
- Animation des résultats
- Design responsive

 Technologies utilisées

- HTML5
- CSS3
- JavaScript (Vanilla)
